# GitHubPage Demo

Đây là một trang tĩnh mẫu để bạn thử giao diện và triển khai lên GitHub Pages.

Hướng dẫn nhanh:

1. Tạo repository trên GitHub (ví dụ: `username.github.io` hoặc bất kỳ repo nào nếu dùng gh-pages branch).
2. Đẩy (push) toàn bộ nội dung trong thư mục này lên repo.
3. Vào Settings -> Pages trong repo GitHub và chọn nguồn (branch) để bật GitHub Pages.

Xem trước cục bộ:

- Nếu có Python 3: `python -m http.server 8000` (chạy trong thư mục gốc) rồi mở http://localhost:8000
- Hoặc dùng extension Live Server trong VS Code.

Chỉnh sửa nhanh: sửa `index.html` và `css/styles.css`.
